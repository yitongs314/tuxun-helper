# 🌍 图寻辅助工具

一个专为图寻玩家设计的街景推理辅助App。  
你可以输入你在街景中看到的细节（语言、车牌、建筑、电线杆等），  
系统将自动高亮可能国家，并排除不符合条件的地区，帮助你快速锁定答案！

## 🧠 支持的线索类型

- 语言（图文辅助 + 国家排除）
- 车牌颜色与形状
- 行车方向（左/右行车自动排除国家）
- 太阳方位（判断南/北半球）
- 电线杆、建筑风格、植被（持续扩展中）
